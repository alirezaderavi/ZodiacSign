# Zodiac Sign Predictor 🪐

This is a simple Python project that takes a user's name and birthdate, determines their Zodiac sign, and stores the result in a CSV file using Pandas.

## 📌 Features

* Input: Name and birthdate
* Output: Zodiac sign
* Data stored in CSV file
* Clean and beginner-friendly code

## 🛠️ Technologies Used

* Python
* Pandas

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/zodiac-sign-app.git
```

2. Navigate into the project:

```
cd zodiac-sign-app
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the program:

```
python ZodiacSign.py
```
